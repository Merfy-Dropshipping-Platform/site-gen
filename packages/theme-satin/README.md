# @merfy/theme-satin

Satin — fashion & clothing theme. Elegant monochrome palette (#000000 / #FFFFFF / #F5F5F5), Kelly Slab headings, Arsenal body, Manrope buttons, UPPERCASE nav/buttons with 0.05em tracking, flat (0px) radii.

Built on top of `@merfy/theme-base` with Header + Footer overrides. Phase 2b migration from legacy `templates/astro/satin/`.

# Vanilla Theme

Warm beige neutral theme for decor and interior stores on Merfy. Uses system-ui fonts with a warm stone/accent palette.

## Features

- Variants, collections, color swatches, filter sidebar, newsletter
- 5 page templates: index, product, collection, cart, blog
- 4 color schemes: Light, Dark, Warm, Accent
- Blog with RSS feed
- Responsive (mobile-first with breakpoints up to 2xl)
